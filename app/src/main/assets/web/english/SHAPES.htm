CIRCLE

A circle is a simple closed shape. It is the set of all points in a plane that are at a given distance from a given point, the centre; equivalently it is the curve traced out by a point that moves so that its distance from a given point is constant.

STAR

is a type of non-convex polygon. Only the regular star polygons have been studied in any depth; star polygons in general appear not to have been formally defined.

 

 

OVAL

(from Latin ovum, "egg") is a closed curve in a plane which "loosely" resembles the outline of an egg. The term is not very specific, but in some areas (projective geometry, technical drawing, etc.) it is given a more precise definition, which may include either one or two axes of symmetry.

 

 

OCTAGON

from the Greek ὀκτάγωνον oktágōnon, "eight angles") is an eight-sided polygon or 8-gon.
